Custom Range Slider

Date Created: 2/5/2021

Date Modified: 2/5/2021

Description: A custom range slider built using HTML, CSS, and Javascript.

Source: @coding_gyan_ from Instagram

Lessons Learned: This exercise introduced me to the idea of setting -webkit-appearance, -moz-appearance, and appearance to none to override default styles set by some browsers. Another issue that I was having trouble was that lines 2-3, which involved .getElementById(), in my customRangeSlider.js file were returning null. I realized that the reason for the error was because the Javascript code was trying to get the element before the page had finished loading. This was resolved by nesting the Javascript code into a window.onload statement.
